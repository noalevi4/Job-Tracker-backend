# Job Tracker Backend

A production-style backend service for managing job applications, companies, interview stages, and follow-up tasks.

## Goals
- Practice backend development with Java and Spring Boot
- Learn clean architecture and REST API design
- Work with PostgreSQL, authentication, validation, testing, and Docker
- Build a project that reflects real-world backend engineering practices

## Planned Features
- User registration and login
- Company management
- Job application tracking
- Task and reminder management
- Filtering, pagination, and search
- API documentation
- Dockerized local setup

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- Maven
- GitHub
